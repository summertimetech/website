The Web

Categorie
CMS
Profiel Web Manager

De Web Manager bedient zich met tools die in staat stellen alle vormen van inhoud op het web te publiceren. Een van de mogelijke tools is een Content Management Systeem. Het is een complexe omgevingen en ontwikkeling geschiedt meestal door speciaal daartoe opgeleide medewerkers. Alhoewel iemand met IT affiniteit "eenvoudige" resultaten kan behalen.

WordPress

WordPress is free open source CMS software en wordt door een grotere groep ontwikkelaars ondersteund. WordPress is het meest gebruikte content management systeem.

→ zie: Download via website WordPress

WordPress maakt gebruik van de programmeertaal PHP en alle content wordt opgeslagen in een MySQL database. Per WordPress installatie is één website te beheren. Voor een meervoudige installatie is WordPress Multisite beschikbaar.

WordPress is een CMS waardoor de gebruiker op een niét technische manier de inhoud (content) kan wijzigen en/of aanvullen. Kennis van coding of software engineering is niet vereist. Door middel van thema's (themes) of zelfgemaakte template files is de opmaak van pagina's aan te passen, zodat er geen gebruik hoeft te worden gemaakt van de standaardvormgeving die WordPress levert.

Ook zijn er talloze plugins beschikbaar die allerlei functies aan het weblog toevoegen. Een aantal daarvan zijn bijvoorbeeld spamfilters, bijvoorbeeld Akismet, kleinschermdetectie voor mobiele apparaten of page builders om de website vorm te geven zonder het gebruik van code of shortcodes. Dit vereist vaak beperkte technische kenniss.

Profiel Web Manager

De Web Manager bedient zich met tools die in staat stellen alle vormen van inhoud op het web te publiceren. Een van de mogelijke tools is een Content Management Systeem. Het is een complexe omgeving en ontwikkeling geschiedt meestal door speciaal daartoe opgeleide medewerkers. Alhoewel iemand met IT affiniteit "eenvoudige" resultaten kan behalen.

WordPress

WordPress is free OpenSource CMS software en wordt door een grotere groep ontwikkelaars ondersteund. WordPress is het meest gebruikte content management systeem.

→ zie: Download via website WordPress

WordPress maakt gebruik van de programmeertaal PHP en alle content wordt opgeslagen in een MySQL database. Per WordPress installatie is één website te beheren. Voor een meervoudige installatie is WordPress Multisite beschikbaar.

WordPress is een CMS waardoor de gebruiker op een niét technische manier de inhoud (content) kan wijzigen en/of aanvullen. Kennis van coding of software engineering is niet vereist. Door middel van thema's (themes) of zelfgemaakte template files is de opmaak van pagina's aan te passen, zodat er geen gebruik hoeft te worden gemaakt van de standaardvormgeving die WordPress levert.

Ook zijn er talloze plugins beschikbaar die allerlei functies aan het weblog toevoegen. Een aantal daarvan zijn bijvoorbeeld spamfilters, bijvoorbeeld Akismet, kleinschermdetectie voor mobiele apparaten of page builders om de website vorm te geven zonder het gebruik van code of shortcodes. Dit vereist vaak beperkte technische kennis.

Drupal

Drupal is free OpenSource CMS software en is het op een na meest gebruikte content management systeem. De standaard release van Drupal, bekend als Drupal core, bevat basisfuncties die gebruikelijk zijn in systemen voor inhoudbeheer. Deze omvatten registratie en onderhoud van gebruikersaccounts, menubeheer, RSS-feeds, taxonomie, aanpassing van de pagina indeling en systeembeheer.

→ zie: Download Drupal via website Drupal

De kerninstallatie van Drupal kan dienen als een eenvoudige website, een blog voor één of meerdere gebruikers, een internetforum of een community website die door gebruikers gegenereerde inhoud biedt. Drupal beschrijft zichzelf ook als een framework. In vergelijking met andere frameworks voldoet Drupal aan de meeste algemeen aanvaarde functievereisten.

Hoewel Drupal een geavanceerde API voor ontwikkelaars biedt, vereisen de basisinstallatie van de website en het beheer van het framework geen programmeervaardigheden. Drupal werkt op elk computerplatform dat zowel een webserver ondersteunt die PHP kan uitvoeren als een database zoals MySQL of PostGress om inhoud en configuratie op te slaan.

In de standaardconfiguratie van een Drupal website kunnen auteurs inhoud toevoegen als geregistreerde of anonieme gebruikers (naar goeddunken van de beheerder). Deze inhoud is toegankelijk voor webbezoekers via verschillende selecteerbare criteria. Vanaf Drupal 8 zijn enkele Symfony bibliotheken in de kern overgenomen.

Joomla

Joomla is free OpenSource CMS software en is het op twee na meest gebruikte content management systeem. Het is gebaseerd op een webapplicatie framework voor model view controller dat onafhankelijk van de CMS kan worden gebruikt.

→ zie: Download via website Joomla

Joomla is geschreven in PHP, maakt gebruik van objectgeoriënteerde programmeertechnieken en software ontwerppatronen, slaat gegevens op in een MySQL, MS SQL of PostgreSQL database en bevat functies zoals paginacaching, RSS-feeds, printbare versies van pagina's, nieuwsflitsen, blogs, zoeken en ondersteuning voor taalinternationalisatie.

Joomla heeft duizenden geverifieerde extensies van derden die te vinden zijn in de Joomla Extensions Directory extensions.joomla.org. Er zijn ook duizenden hoogwaardige sjablonen beschikbaar, waarvan vele gratis zijn. Betaalde sjablonen zijn ook beschikbaar en worden geleverd met ondersteuning.

Veel sjablonen bieden een grafische gebruikersinterface waarmee u kleuren, lettertypen, lay-outs en functies kunt wijzigen. Joomla heeft ingebouwde SEO-tools. Inclusief metadata en trefwoorden, Mod_rewrite-ondersteuning voor SEF-URL's en menu-creatie voor duidelijke en consistente sitemaps.
